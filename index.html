Dom Play

A Javascript Example by Kamari Senters

This is a web page that uses JavaScript for a single page application using DOM. The page will:

  1. With a single click, highlights all lines for a specific actor in a play
  2. Un-highlight (normal or otherwise) all lines from all other actors in the play
  3. The function(s) created to do the work must accommodate any number of actors, and additional lines added to the page
  4. The page must do the work without page re-loads

Hamlet: To be or not ot be?

Skull: Are you talking to me?

Hamlet: Did yoou say something?

Skull: No, I'm only a skull.

Ophelia: Hamlet! Stop taling to skulls.
